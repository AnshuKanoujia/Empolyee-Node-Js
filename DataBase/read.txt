INSTRUACTION TO SETUP THE PROJECT.

1.) INSTALL NODE, NPM & MYSQL.
2.) CLONE THE WHOLE PROJECT ON YOUR DIRECTORY.
3.) SETUP THE DATA-BASE WITH NAME "emp_tracker".
4.) MAKE THE SUITABLE CHANGES IN /config /db.js BY PROVIDING MYSQL USER & PASSWORD.
5.) RUN NPM UPDATE COMMAND IN PROJECT DIRECTORY FROM TERMINAL.
6.) THEN RUN NPM AUDIT FIX COMMAND.
7.) FINALY RUN NPM RUN SERVER COMMAND.
8.) THEN PROCCED TO YOUR BROWSER, YOU WILL BE FIND YOUR PROJECT RUNNING ON http://YOUR-IP:4000.
